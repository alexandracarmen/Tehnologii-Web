Abstract

1.Purpose

This application provides support to people who want to share their recipes with other people who have the same culinary preferences. In the vegetarian group we have recipes for vegetarians, as well as in the other groups for vegans and the family group.

Registered User

  • Log in to the application using email and password.After logging in, he has the following rights:
    • has access to user groups
    • has access to all recipes
    • the user can add a recipe
    • the user has a favorite recipe
    • has access to all social links on other platforms

Users without an account

  • By entering the website, he has the following rights:
    • he can sign up by : entering his first and last name, entering his valid e-mail, an username and password
    • can access all the information present on the web site divided into categories:general recipes, available foods and information about them

Admin user

  • By passing all the tests on the web site, this kind of users (admin users) have additional rights:
    • has an additional right: can see the last people authenticated on the site, email addresses, as well as the date of authentication.
    • can change the number of users we can see
    • contains the products, users and returns us to the main page
